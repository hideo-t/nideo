第1章

量産者

深夜の静寂。街には遠くから車の走る音が聞こえてくるだけだった。小さな部屋の中、蛍光灯ではなくモニターの青白い光だけで照らされている空間がある。その中で一人、男は机に向かい、パソコンを操作している。

雨粒が窓ガラスに落ちて静かな曲線を作り出す音。アパートの一室から漏れる生活音。この時間帯ならまだ誰も起きていないはずだが、彼の隣には黒い猫のような影があるように思える。それは彼にとってAIたちだ。

コーヒーの液がカップの中で渦を巻く様子は見ることもなく、ただその匂いだけが部屋に漂っている。男は眼鏡の奥で画面を見つめ、キーボードを素早く叩き続ける。

ターゲット：新アプリ開発。

ステップ1：要件定義。

秀夫と呼ばれる彼の指先から次々と命令が出され、それがターミナル上で文字列として現れ、すぐにまた消えていく。速さだけが価値となる世界で、彼はその中心にいる一人の人間だった。

ステップ2：設計。

git clone project-template

cd app-new

自動化パイプラインが唸りを上げる音が耳元から響く。コードの行数が増え、グラフィックデザイナーが出したイラストも並んでいく。Ollamaは文章をつむぎ出し、Stable Diffusionは絵を作っていく。

ステップ3：開発。

npm install

yarn build

10章構成、挿絵付きの物語が完成するまでの時間はたった12分だった。その間に彼は何度もコーヒーを淹れ、カップから口元へと運んでいく。味わうわけでもなくただ飲むだけだ。

「また、できた。」

それは自問のような言葉で響き渡る。新しいリポジトリがGitHubにpushされると同時に通知音が鳴り止める。彼は瞬間的に満足感を得た後に、その表情を改めてしまう。

静けさと光の狭間に座る男。

画面を見つめたまま、秀夫は何度も手元からコーヒー杯を取り上げてはまたそれを下す動作を繰り返した。

「速さは、ある。でも——」

彼が思わず呟いた言葉は、その後すぐに消えてしまうようだった。それはただの影のように見え、しかし音として残るだけだ。

画面を閉じた瞬間、光と闇との境界線上で一瞬手元を見つめた。

「これは本当に、刺さっているのか？」
